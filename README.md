# decentralizedma.com

Marketing website for **Decentralized Machine Automation (DMA)** — engineering-led AI automation for electrical service companies in Houston and beyond.

**Tagline:** Simplify. Automate. Execute.
**Contact:** jescobar@decentralizedma.com

## Stack

Plain static site — hand-written HTML + CSS + vanilla JS. No framework, no build step.

## Local preview

```bash
open index.html
```

Or serve with any static server:

```bash
python3 -m http.server 8000
```

## Structure

- `index.html` — single-page site (hero, problem, services, process, about, proof, CTA, footer)
- `styles.css` — all styles, brand tokens at `:root`
- `assets/fonts/` — Instrument Sans (headings), Work Sans (body)
- `assets/images/` — logo + favicon
- `assets/video/` — hero background video (mechanical → digital gears)
